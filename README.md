# Personal Portfolio Website | Frontend Project

---

## About
This is a **fully functional and responsive personal portfolio website** developed using **HTML, Tailwind CSS, and JavaScript**. It presents my professional profile, technical skills, projects, and contact information in a clean, modern, and interactive layout.

---

## Highlights
- **Mobile-first design** using Tailwind CSS  
- Fully **responsive across all devices**
- Smooth scrolling and dynamic interactions using JavaScript  
- Well-structured sections: About, Skills, Projects, and Contact  
- Clean and modern UI for a professional appearance
